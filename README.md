# 🌤️ Weather Dashboard (Previsão do Tempo)

Um painel de previsão do tempo moderno e interativo construído com **Next.js** e **React**. Este projeto consome os dados climáticos em tempo real através da API da [HG Brasil](https://hgbrasil.com/) e exibe as condições atuais, detalhes climáticos, destaques do dia e a previsão para os próximos dias.

## 🚀 Tecnologias Utilizadas

* **[Next.js](https://nextjs.org/)** - Framework React (utilizando o moderno App Router).
* **[React](https://reactjs.org/)** - Biblioteca JavaScript para construção de interfaces.
* **API HG Brasil** - Fornecimento dos dados meteorológicos.

## 📁 Estrutura do Projeto

O projeto utiliza a nova estrutura de rotas do Next.js (App Router):
* `src/app/` - Contém as páginas principais e rotas do aplicativo (`page.js`, `layout.js`).
* `src/components/` - Contém todos os componentes da interface (ex: `Header`, `CurrentWeather`, `CitySearch`, etc.).

## 🛠️ Como executar o projeto na sua máquina

### Pré-requisitos
Antes de começar, certifique-se de ter o **[Node.js](https://nodejs.org/)** instalado no seu computador.

### Passo a passo

1. **Abra o terminal** na pasta do seu projeto.

2. **Instale as dependências** executando o comando abaixo:
```bash
   npm install
```

3. **Inicie o servidor de desenvolvimento:
```bash
  npm run dev
```

4. **Acesse no navegador:
```bash
  http://localhost:3000
```
